How to keep your Gmail Account Secure?

Like every other email servers, Google’s systems are also prone to cyber attack. To neutralize
these actions, one has to stay updated with the security features and suggestions provided by
Gmail.
Among the few easiest steps in this regard, the preliminary one is not to disclose your login
credentials. Any emails or text inquiring such details should be ignored immediately. Any email
servers including Gmail would never request such information explicitly form users. Further one
should be in practice of changing passwords frequently. Gmail too recommends changing your
password twice a year at least without repeating the previous passwords in the span of six
months.

Security

Password security also demands non repetitive usage of the same password on multiple websites. For instance, using same password on Gmail along with some other website increases the chance of its discovery. It is for the reason, that the other website may not follow the high security measures as advance email servers do and password once revealed from that system may also be used by the hacker for login trials on your Gmail web account.


Strong email password should be created using letters and the numbers both. After creating the password one should neither make a note of it nor should it be mailed to self. Actually, this turns out a very common mistake. If it is difficult to remember, put down a clue for yourself which should not hold any segment of the password.
Keeping your system (PC) secure is incredibly crucial. Updates are persistently released by the vendors of operating systems and web browsers for keeping them secure. Running some antivirus software ensures the security of your system software even if your email server is under some security threat.


Make sure you update your Gmail account instantly if you make any change in your secondary Email address or in your security question. Accuracy of this information will ensure easiness in recovering your web account.


As the previously used protocol “HTTP” was less secure, one should always choose the option of using “HTTPS” connection while setting up the Gmail account. This prevents the hijacking of personal information while accessing from public computers. Also, while using a public network or computer, like in cyber café, one must ensure that he has completely logged out the account before leaving.
If at any instant you notice any unidentified changes or requests in your account, right away modify the entire log in credentials being used at that time. Recurrent scans should be run. There are various free of cost malware detection and anti-spyware software available over the internet. Using such software will make it easier for you to discover any threat over your system.
