How to create EC2 instance on AWS Free Tier plan.

This blog is in continuation of HOW TO CREATE AWS FREE TIER ACCOUNT. if you have missed it please revisit.

Once your AWS account is active you can see following screen after login.

1. AWS Services List

Go to services link at top.  once you click on services link you will be able to see following menus :

2. EC2 dashboard Page

Click on EC2 link to open EC2 dashboard. once you click on EC2 link you will be able to see following dashboard page.

3. Launch Instance

You can see there is 0 Running instances . Now click on Running instances to see all instances if they exist already. once you click on Launch instance you should be able to see following page :

4. Select AMI machine (i.e EC2 Ubuntu ) for Launching

Click on Launch Instance Button to launch your first EC2 instance ( free tier ) you will be redirected to following page. click on Ubuntu Server 16.04

5. Creating AMI machine - EC2 Instance

After you select the AMI machine type you will be asked for following configurations . select t2.micro ( free tier eligible) instance

6. Creating Private/Public Key pair for SSH

You will be asked to create a new key pair. this you will be using to connect to your EC2 instance through SSH. create and download the key pair in your system.

7. View EC2 Instances After Successfull Launch

After creating and downloading your key. click on Launch Instances. you will be able to see following screen. which confirms that your EC2 instance is now running and you can check on View Instances.

8. Edit Security Group

Click on View Instances. you will be able to see the Instance details go to Security Groups to change the security settings for the instance. (Beware : only make those ports public which you want to.) :

9. AWS Security Group

By default SSH is enabled in security group. you can edit as per your need.t (see below image for reference).
