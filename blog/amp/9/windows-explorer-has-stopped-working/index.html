Windows explorer has stopped working?

Since the beginning of the windows era there have been few windows’ errors which manage to show their existence every now and then. A very common among them that would come into your mind would be the critical recovery error shown by windows, however if you go for searching about it in detail then you would land among numerous entries. Similarly, another very common error is "Windows explorer has stopped working".
This one is very frequent on windows 7 enabled PC. The reason for its recurrent occurrence is that they have multiple causes. It’s not some single issue which brings such error, hence it’s hard to fix. Here, we will list out few steps to be used in order to fix such errors.

Clean Boot Testing

In search for the prime reason you’ll come across the fact that it was caused by some application other than OS, as windows rarely corrupts itself. So one should strive hard to isolate the program, startup or any a service which is generating the issue.
It can be done by performing the clean boot as stated below:
Firstly, all third party services and items under startup should be disabled using the MSCONFIG and look if the problem goes away. A positive result indicates that either it is a service or may be a startup item. Now you need to reboot the system again and again enabling half the applications. Though it is bit time consuming but worth, as in the end it would leave your computer working properly.
 

Step 2 -- Disable Shell Extensions
If problem persists even after the clean boot process then all non Microsoft shell extensions should be halted. This may be done with the help of a free plug-in from Nirsoft known as ShellExView. 
First execute the program sorting them by the name of manufacturer, and then items other than from Microsoft should be stopped. Now once again if the issue is resolved it means that it was because of any one of the shell extension, which you may check enabling them one by one. In order to disable any item simply click the red button after selecting it.
 

Step 3 – Updating the Video Driver
In some rare cases, a video driver that is old or might have been infected, may also cause this problem. You may simply go to your hardware vendor’s site and download its latest 32 or 64 bit compatible version for windows.

Step 4 – Selection of Always show icon box
One more solution would be selecting the Always show icon, never thumbnail box in the View tab in the Windows explorer, for Windows XP (Tools > Folder Option > View), and for Windows Vista or 7 (Organize > Folder and search option > View). 
This step would ensure that your system always shows an icon view of your image and video files instead of a small thumbnail picture. Although it may not appear as a direct solution but possibly your problem may get resolved and no more error message stating “Windows explorer has stopped working” would come.
 

Step 5 – Restoring the System
If you are facing the problem after installing some new updates or any new application then it is recommended that you shall restore your computer system. System restore can be done by simply typing “system restore” keyword in start menu. It ensures that your system reaches to a better state, a time before the installation was done.
 

Step 6 – Run the System File Checker command
Although rare, but there exist a possibility that this problem might have been caused due to corrupted files. You may resolve it using several tools. To start with, go to the command prompt and run the system file checker command. In order to open the command prompt in admin mode go to the start menu and type the “command prompt” keyword, and then select run it as Administrator. Lastly, type the following listed command:

sfc /scannow


Method 7 – Repair Install Windows
Another step could be repairing your Windows. For this go on to the startup repair from the System Recover Option. All you need to do is press F8 button while your system is booting. From the list of Advanced Boot Options, select Repair your computer.
Run the Startup Repair by selecting it. Further while you are in the recovery option of your system, you may select the command prompt there and try running this command:

chkdsk c: /r


Step 8 – Removing the Virus and Malwares 
If none of the above steps had helped you out in getting rid of “Windows explorer has stopped working” error, it means your system is infected with some malwares or a virus. Go for installation of some antivirus software and anti-spywares and check if your problem fixes up. In case of heavy virus attack you may be left with the only option of reinstalling your windows after formatting.
